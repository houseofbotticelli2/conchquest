Conditions & Score

    

      ?
      
Conditions pulls together tide, wind, wave, weather, and moon-phase data for a lat/lon and normalizes it into one response (cached server-side). Score uses that same data to compute a 0–100 Shelling Score, a confidence level, a best time-of-day window, and a plain-English explanation.

    

  

  

    

      
      
    

    

      
      
    

  

  
  

    
    
  

  

  






  

    
Log a Find

    

      ?
      
POST records a shell find (location, condition, notes) tied to whoever's logged in. GET /api/finds lists your own logged finds. GET /api/finds/:id looks up one specific find — copy an id from the list result first.
